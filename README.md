# Veloce / Vogue Motors

Plateforme de vente de véhicules avec authentification sécurisée et paiement en ligne.

## Stack
- Node.js, SQLite
- Google OAuth (PKCE)
- FedaPay

## Fonctionnalités
- Authentification sécurisée via Google OAuth
- Intégration paiement FedaPay pour l'achat de véhicules
- Backend custom (SQLite/node:sqlite)
